# Projet de Gestion Financière

Ce projet est une analyse financière à court terme réalisée avec Python et Jupyter Notebook. Il vise à démontrer mes compétences en programmation et en gestion de données.

## Contenu du projet
1. `Gestion Financiere a court terme_Livre.ipynb` :
   - Un fichier Jupyter Notebook contenant l'analyse financière principale.
2. `Introduction_Au_Livre.py` :
   - Un script Python pour compléter l'analyse.
3. `requirements.txt` :
   - Une liste des bibliothèques Python nécessaires pour exécuter le projet.
